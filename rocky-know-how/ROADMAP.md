```markdown
## [DEC-20260420-001] 使用Redis还是Memcached？

**问题**: 需要为WordPress选择缓存方案
**选项**:
  - A. Redis - 功能丰富，支持多种数据结构
  - B. Memcached - 简单高效，纯内存
**决策**: Redis
**原因**: 
  - 支持持久化
  - 社区活跃
  - 主站已在用，减少维护成本
**日期**: 2026-04-20
**决策者**: 大正
**状态**: 有效
```

### 搜索升级路径

```
v2.5: 关键词 + 向量
v3.0: 关键词 + 向量 + 语义理解
v4.0: 关键词 + 向量 + 语义 + 知识图谱
```

### AI判断流程（v3.0）

```
用户说"解决了XXX"
       ↓
Hook检测 → 写入pending
       ↓
AI判断（GPT/本地模型）
       ↓
┌─────────────────────────────────┐
│ 判断结果:                        │
│ - 全新经验 → 新建经验卡           │
│ - 已有经验补充 → 更新版本         │
│ - 通用知识 → 归类到API/项目       │
│ - 无价值 → 归档或丢弃            │
└─────────────────────────────────┘
       ↓
更新知识库 + 更新图谱关系
```
